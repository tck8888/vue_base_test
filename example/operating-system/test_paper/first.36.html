请分别从资源分配、创建速度、通信、并行执行效率的角度筒述多线程技术的优势。

  1. 创建线程无需外分配资源,而创建进程需要分配资源;
  2. 创建线程无需分配资源,因而创建线程速度比创建进程速度快
  3. 线程间的通信在同一地址空间中进行,不需要额外的通信机制,所以通信简单,信息传递速度也更快
  4. 线程能独立进行，充分利用和发挥处理器与外围设备并行工作的能力

解释通道命令、通道程序、通道地址字和通道状态字。

  1. 通道命令:规定了设备一种操作的命令
  2. 通道程序:若干条通道命令组成的程序,有通道执行,完成一次I/O操作
  3. 通道地址字:用来存放通道程序首地址的主存固定单元
  4. 通道状态字:用来记录通道程序执行结果的主存固定单元
